# Security Policy

## Supported Versions

Only the latest release receives security fixes.

## Reporting a Vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities privately by emailing **bilalgunay@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (optional)

You will receive a response within 72 hours. Once the issue is confirmed and a fix is released, you will be credited in the release notes (unless you prefer to remain anonymous).

## Scope

- The Flutter app source code
- Supabase Edge Function logic
- Keyword matching engine

Out of scope: third-party services (Supabase, OpenFoodFacts, Anthropic API).
